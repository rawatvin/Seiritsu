## 🔑 Key Talking Points

### When Management Asks About Mobile:

**Q: "Does mobile increase the cost?"**
A: No! Mobile web browser is included in responsive design ($0). Native iOS/Android apps are also free with Power Platform ($0). The cost difference between Lean ($25k) and Standard ($84k) is due to implementation scope and premium features, NOT mobile.

**Q: "What mobile access do we get?"**
A: 
- Lean: Mobile web browser (works great, add to home screen)
- Standard: Same, PLUS native iOS/Android apps
- Both: $0 additional cost

**Q: "Can we upgrade later?"**
A: Yes! Start Lean ($25k-50k), if users want native apps after 3 months, enable them for $0 in 1 week.

**Q: "Why choose Standard over Lean?"**
A: If you want ALL features from Day 1 (native apps, offline mode, JIRA auto-sync, advanced analytics, full training). NOT because of mobile specifically.

**Q: "What's the minimum we can do?"**
A: Lean approach: $25k-50k, includes mobile web browser access, works on all phones, 500 users, 12 weeks.
