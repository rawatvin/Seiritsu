✅ Access from desktop and mobile devices

## 📊 What's Included (Lean)

### Core Features ✅

**Portfolio Management:**
- Submit requests via web form
- Track requests through 5 stages
- Comment, attach files, watch items
- Search and filter
- Stage management (Kanban board)

**SLA Tracking:**
- Automatic calculation per stage
- Color-coded alerts (Green/Yellow/Red)
- Email notifications for breaches
- Escalation to leads after 24h

**Dashboards:**
- "My Assignments" (personalized)
- "Watching" (items you follow)
- "Team View" (department/market)
- SLA status overview

**Reporting:**
- Basic charts (requests by stage, SLA compliance)
- Export to Excel
- Simple trend analysis

**Mobile Access:**
- **Responsive web browser** (works on any phone)
- Touch-friendly interface
- Add to home screen (icon on phone)
- Desktop + mobile + tablet support

**Security:**
- Azure AD authentication (company credentials)
- Role-based access control
- Audit logs

### Not Included (Can Add Later) ⏸️

**Native Mobile Apps** (optional, $0 to enable)
- Power Apps Mobile for iOS/Android
- Offline mode
- Push notifications
- *Add if users request it after launch*

**JIRA Auto-Sync** (optional, $500/month)
- Start with manual JIRA linking
- Automate later if tedious
- *Most teams fine with manual*

**Advanced Analytics** (optional, $5k/month)
- Basic reports sufficient for start
- Upgrade if C-level demands it
- *Evaluate after 6 months*

**Copilot AI** (optional, $30/user/month)
- Not essential for launch
- Add in 12+ months if desired
- *Nice-to-have*
