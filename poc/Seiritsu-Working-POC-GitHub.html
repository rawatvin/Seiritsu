Excel Import Instructions

Prepare an Excel file (.xlsx or .xls) with the following columns:

Required columns (exact names):

  • request - Request title
  • description - Problem / business case
  • ballparkValue - Annual value in € (number)
  • requesterName - Name of requester
  • requestCategory - Category (New Feature, Enhancement, etc.)
  • market - Market (Global, EMEA, APAC, etc.)
  • countries - Countries in scope
  • globalBusinessLead - Business lead name
  • globalTechLead - Tech lead name
  • status - Status (Submitted, Under Review, etc.)
  • globalComments - Comments
  • marketSuccessOwner - Market success owner
  • productArea - Product area/home
  • capabilityId - Capability ID
  • fundingSource - Funding source (Global, Market, Mixed)
  • linkedMustLand - Linked must land
  • estimatedCapitalCost - Capital cost in € (number)
  • estimatedOperatingCost - Operating cost in € (number)
  • costEstimationDetails - Cost details
  • expectedStartDate - Start date (YYYY-MM-DD)
  • expectedEndDate - End date (YYYY-MM-DD)
  • actualStartDate - Actual start (YYYY-MM-DD)
  • actualEndDate - Actual end (YYYY-MM-DD)

Tip: Export data first to see the exact format, then use it as a template!
