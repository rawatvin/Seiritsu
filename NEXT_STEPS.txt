  🚀 NEXT: PUSH TO GITHUB (5 minutes)
================================================================================

STEP 1: Create GitHub Repository
  → Go to: https://github.com/new
  → Name: task-intelligence-app
  → Visibility: Private (recommended)
  → DON'T initialize with README/gitignore (we have them)
  → Click "Create repository"

STEP 2: Push Your Code
  Open PowerShell and run:

  cd C:\Users\rawatvin\task-intelligence-app

  git config --global user.name "Your Name"
  git config --global user.email "your.email@example.com"

  git remote add origin https://github.com/YOUR_USERNAME/task-intelligence-app.git
  git push -u origin main

STEP 3: Open in Codespaces
  → Go to your GitHub repository
  → Click green "Code" button
  → Click "Codespaces" tab
  → Click "Create codespace on main"
  → Wait 3-5 minutes for setup

STEP 4: Configure API Keys
  In Codespaces, edit: backend/.env
  Add your:
    - Microsoft Azure AD credentials
    - Anthropic API key
    - Update redirect URI with Codespace URL

STEP 5: Start the App
  Terminal 1: cd backend && uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
  Terminal 2: cd frontend && npm run dev -- --host

STEP 6: Open Frontend
  → Click PORTS tab in VS Code
  → Find port 5173
  → Click globe icon 🌐
  → Your app opens in browser!
