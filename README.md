## 📋 What is Seiritsu?

Seiritsu (成立 - Japanese for "establishment/formation") is a comprehensive portfolio management tool that enables Adidas teams to:

- ✅ Submit and track requirements from market/global teams
- ✅ Manage requests through lifecycle stages (Intake → Feasibility → Funding → Roadmap)
- ✅ Monitor SLA compliance with automatic escalation
- ✅ Visualize roadmaps and portfolios
- ✅ Generate personalized dashboards per user
- ✅ Integrate with JIRA, Power BI, and Microsoft Copilot
- ✅ **Native mobile apps (iOS & Android) with offline mode** 📱

---

## 🛠️ Technology Stack

**Recommended Platform:** Microsoft Power Platform
- SharePoint/Dataverse (database)
- Power Apps (frontend)
- Power Automate (workflows & SLA monitoring)
- Power BI (analytics & reporting)
- Microsoft Copilot (AI integration)

**Why Power Platform?**
- Already licensed with Microsoft 365
- Native integration with Teams, Outlook, JIRA
- **Native iOS/Android mobile apps included (no extra development!)**
- **Offline mode & push notifications built-in**
- Enterprise security (Azure AD)
- Low cost & maintenance
- 3-month implementation timeline
