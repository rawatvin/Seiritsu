## 📋 What is Seiritsu?

Seiritsu (成立 - Japanese for "establishment/formation") is a comprehensive portfolio management tool that enables teams to:

- ✅ Submit and track requirements from market/global teams
- ✅ Manage requests through lifecycle stages
- ✅ Visualize roadmaps with interactive timeline
- ✅ Generate personalized dashboards per user
- ✅ Track planned vs actual timelines
- ✅ Filter and analyze by multiple dimensions
